# webЯcade Application: Sega Saturn (Experimental)

[![License](https://img.shields.io/badge/License-Apache%202.0-blue.svg)](https://opensource.org/licenses/Apache-2.0)

The Sega Saturn application for [webЯcade](https://www.webrcade.com) is an emulator for the [Sega Saturn](https://en.wikipedia.org/wiki/Sega_Saturn) game console.

The underlying emulator is based on the Libretro Yabause core.

<p align="center">
 <a href="https://play.webrcade.com">
  <img src="https://docs.webrcade.com/assets/images/apps/saturn.png" width="90%">
 </a>
</p>

> ⚠️ **Experimental**
>
> This application is currently experimental due to high resource requirements and game compatibility issues.  
> It is hidden by default and must be enabled via advanced settings in the player or editor.

### Documentation

The [Sega Saturn Application Documentation](https://docs.webrcade.com/apps/emulators/saturn/) includes keyboard and gamepad control mappings, BIOS configuration, feed setup, and additional usage details.
